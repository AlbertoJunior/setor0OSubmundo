# Índice de Documentação - v13

Esta pasta contém a base da API oficial do Foundry VTT versão 13, utilizada para consultas de classes, métodos, propriedades e interfaces do sistema. A documentação foi pré-processada para remover scripts e estilos irrelevantes, facilitando a indexação e a leitura rápida pelo Agent.

| Arquivo Principal | Descrição | Última Atualização |
| :--- | :--- | :--- |
| [modules.html](./modules.html) | Ponto de entrada (root) principal para a API Documentada do Foundry V13. Ele lista as demais Classes e Namespaces disponíveis. | 08/03/2026 |

*Nota: Para explorar classes específicas, o modelo do Agent fará bucas diretas (`find_by_name` ou `grep_search`) dentro de `interfaces/`, `classes/`, `types/` presentes nesta pasta, com base nos termos identificados através do `modules.html`.*
